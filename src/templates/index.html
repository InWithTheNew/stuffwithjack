Hey Jack. If you've hit this, great news, you're starting to see the value in having APIs.
I assume you're testing, or trying to test, the functions in the app.

If you're trying to hit the get_random_animal function, you can do so by hitting the following endpoint:

https://stuffwithjack{{ branch_name }}.{{ domain_name }}/get_random_animal

If you're trying to hit the get_animal, it's a bit more as it's a POST request (go and learn!).
Use Postman to submit a post request to

https://stuffwithjack{{ branch_name }}.{{ domain_name }}/get_animal

with the following JSON body:

{ "type": "dog", "name": "Buster" }
